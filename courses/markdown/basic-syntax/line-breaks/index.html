Переносы строк

Курсы / Markdown / Базовый синтаксис

В Markdown перенос строки (Line Breaks) может быть не так прост, как в обычных текстовых редакторах. В этом уроке мы рассмотрим, как использовать переносы строк для форматирования текста в Markdown.

Добавление переноса строк

Чтобы добавить простой перенос строки (без создания нового абзаца), в конце строки вставьте два пробела перед переводом строки.

Пример (Markdown):

Этот портал  
открылся совсем  
неожиданно.

Результат (HTML):

Этот портал<br>
открылся совсем<br>
неожиданно.

Результат (Отображение):

Этот портал
открылся совсем
неожиданно.

 

Если ваше приложение для работы с Markdown поддерживает HTML, вы можете использовать тег HTML <br> для создания переноса строки внутри параграфа. 

Пример (Markdown):

Этот портал<br>
открылся совсем<br>
неожиданно.

Результат (HTML): 

Этот портал<br>
открылся совсем<br>
неожиданно.

Результат (Отображение): 

Этот портал
открылся совсем
неожиданно.
