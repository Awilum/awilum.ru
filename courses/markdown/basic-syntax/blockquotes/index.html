Блок цитат

Курсы / Markdown / Базовый синтаксис

Блок цитат (Blockquote) в Markdown позволяет выделить текст, придав ему вид цитаты.

Простой блока цитат

Для создания блока цитат используйте символ большой стрелки > перед текстом цитаты.

Пример (Markdown):

> Это странное место, Хоукинс.

Результат (HTML):

<blockquote>Это странное место, Хоукинс.</blockquote>

Результат (Отображение):

Это странное место, Хоукинс.

Блок цитат с несколькими абзацами

Блок цитат может содержать несколько абзацев текста.

Пример (Markdown):

> Я потерялся в изнанке.  
> На меня охотится что-то странное.

Результат (HTML):

<blockquote>Я потерялся в изнанке.<br>На меня охотится что-то странное.</blockquote>

Результат (Отображение):

Я потерялся в изнанке.
На меня охотится что-то странное.

Вложенные блоки цитат 

Блоки цитат могут быть вложенными.

Добавьте >> перед абзацем, который хотите вложить.

Пример (Markdown):

> Джим: В Лаборатории Хоукинс  
> происходит что-то странное.
>> Уилл: Да, тут какие-то порталы  
>> и демогоргоны!

Результат (HTML):

<blockquote>
	<p>Джим: В Лаборатории Хоукинс<br>происходит что-то странное.</p>
	<blockquote>
		<p>Уилл: Да, тут какие-то порталы<br>и демогоргоны!</p>
	</blockquote>
</blockquote>

Результат (Отображение):

Джим: В Лаборатории Хоукинс
происходит что-то странное.

Уилл: Да, тут какие-то порталы
и демогоргоны!

Блоки цитат с другими элементами 

Блоки цитат могут содержать другие элементы в формате Markdown.

Не все элементы можно использовать — вам придется поэкспериментировать, чтобы увидеть, какие из них работают.

Пример (Markdown):

> __Эл__: Я могу видеть вещи,  
> которые другие не видят.
>
> **Дастин**: Так, Эл, давай лучше поиграем  
> в Dungeons & Dragons.

Результат (HTML):

<blockquote>
	<p><strong>Эл</strong>: Я могу видеть вещи,<br>которые другие не видят.</p>
  	<p><strong>Дастин</strong>: Так, Эл, давай лучше поиграем<br>в Dungeons &amp; Dragons.</p>
</blockquote>

Результат (Отображение):

Эл: Я могу видеть вещи,
которые другие не видят.

Дастин: Так, Эл, давай лучше поиграем
в Dungeons & Dragons.
