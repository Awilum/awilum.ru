Ссылки

Курсы / Markdown / Базовый синтаксис

Ссылки (Links) в Markdown позволяют вставлять гиперссылки для перехода к другим веб-ресурсам или документам.

В Markdown существует несколько способов создания ссылок, а также дополнительные возможности форматирования, которые могут сделать их более выразительными.

Основы создания ссылок

Создание ссылок осуществляется с использованием квадратных скобок для отображения текста ссылки и круглых скобок для указания самой ссылки.

Пример (Markdown):

[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things)

Результат (HTML):

<a href="https://en.wikipedia.org/wiki/Stranger_Things">Очень странные дела</a>

Результат (Отображение):

Очень странные дела

Ссылки с заголовками (Titles)

Вы можете добавить заголовок для ссылки, который появится в виде подсказки, когда пользователь наводит курсор на ссылку. Для этого заключите заголовок в кавычки после URL.

Пример (Markdown):

[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things "Американский научно-фантастический драматический сериал ужасов, созданный братьями Даффер для стриминг-сервиса Netflix")

Результат (HTML):

<a title="Американский научно-фантастический драматический сериал ужасов, созданный братьями Даффер для стриминг-сервиса Netflix" href="https://en.wikipedia.org/wiki/Stranger_Things">Очень странные дела</a>

Результат (Отображение):

Очень странные дела

Форматирование ссылок

Чтобы выделить или изменить стиль текста внутри ссылки, используйте звездочки или подчеркивания.

Пример (Markdown):

Мне нравится **[Stranger _Things_](https://en.wikipedia.org/wiki/Stranger_Things)**.

Результат (HTML):

Мне нравится <strong><a href="https://en.wikipedia.org/wiki/Stranger_Things">Stranger <em>Things</em></a></strong>.

Результат (Отображение):

Мне нравится Stranger Things.

Ссылки на URL и Email

Чтобы быстро создать ссылку на URL или адрес электронной почты, заключите их в угловые скобки.

Пример (Markdown):

<https://en.wikipedia.org/wiki/Stranger_Things>
<eleven@strangerthings.com>

Результат (HTML):

<a href="https://en.wikipedia.org/wiki/Stranger_Things">https://en.wikipedia.org/wiki/Stranger_Things</a>
<a href="mailto:eleven@strangerthings.com">eleven@strangerthings.com</a>

Результат (Отображение):

https://en.wikipedia.org/wiki/Stranger_Things
eleven@strangerthings.com
