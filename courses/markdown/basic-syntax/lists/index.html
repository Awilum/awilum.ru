Списки

Курсы / Markdown / Базовый синтаксис

Списки (Lists) в Markdown позволяют организовать информацию в упорядоченные и неупорядоченные структуры.

Давайте рассмотрим, как создавать списки и добавлять элементы.

Неупорядоченные списки

Неупорядоченные списки (Unordered Lists) создаются с использованием символов -, * или +.

Пример (Markdown):

- Глава первая: Исчезновение Уилла Байерса
- Глава вторая: Чудак с Кленовой улицы
- Глава третья: Здорово, Джолли

Результат (HTML):

<ul>
	<li>Глава первая: Исчезновение Уилла Байерса</li>
	<li>Глава вторая: Чудак с Кленовой улицы</li>
	<li>Глава третья: Здорово, Джолли</li>
</ul>

Результат (Отображение):

  • Глава первая: Исчезновение Уилла Байерса
  • Глава вторая: Чудак с Кленовой улицы
  • Глава третья: Здорово, Джолли

Упорядоченные списки

Упорядоченные списки (Ordered Lists) создаются с использованием числовых значений и точек.

Пример (Markdown):

1. Джойс Байерс
2. Хоппер
3. Одиннадцать

Результат (HTML):

<ol>
	<li>Джойс Байерс</li>
	<li>Хоппер</li>
	<li>Одиннадцать</li>
</ol>

Результат (Отображение):

  1. Джойс Байерс
  2. Хоппер
  3. Одиннадцать

Добавление элементов в списки

Элементы списка могут включать в себя другие элементы, текст, изображения или даже блоки цитата и кода.

Пример (Markdown):

- **Джойс Байерс**
- Уилл
- Джонатан
- **Хоппер**
- **Одиннадцать**
- Майк
- Лукас
- Дастин

Результат (HTML):

<ul>
	<li>
		<strong>Джойс Байерс</strong>
		<ul>
		<li>Уилл</li>
		<li>Джонатан</li>
		</ul>
	</li>
	<li><strong>Хоппер</strong></li>
	<li>
		<strong>Одиннадцать</strong>
		<ul>
		<li>Майк</li>
		<li>Лукас</li>
		<li>Дастин</li>
		</ul>
	</li>
</ul>

Результат (Отображение):

  • Джойс Байерс
    • Уилл
    • Джонатан
  • Хоппер
  • Одиннадцать
    • Майк
    • Лукас
    • Дастин
