Заголовки

Курсы / Markdown / Базовый синтаксис

В Markdown заголовки (Headings) используются для выделения различных уровней заголовков или разделов в тексте. Заголовки в Markdown создаются с использованием символов решетки #. Количество символов # определяет уровень заголовка, и чем меньше #, тем выше уровень заголовка.

Заголовок первого уровня создается с использованием одной решетки

Пример (Markdown):

# Очень странные дела

Результат (HTML): 

<h1>Очень странные дела</h1>

Результат (Отображение):

Очень странные дела

Заголовок второго уровня создается с использованием двух решеток

Пример (Markdown):

## Главы

Результат (HTML): 

<h2>Главы</h2>

Результат (Отображение):

Главы

И так далее, до заголовка шестого уровня.

Пример (Markdown):

###### Майк

Результат (HTML): 

<h6>Майк</h6>

Результат (Отображение):

Майк

Альтернативный способ для заголовков первого и второго уровней

Пример (Markdown):

Очень странные дела
========================

Главы
------------------------

Результат (HTML): 

<h1>Очень странные дела</h1>

<h2>Главы</h2>

Результат (Отображение):

Очень странные дела
Главы

 

Старайтесь подбирать уровень заголовка в соответствии с логикой структуры вашего текста, чтобы создать четкую и легко читаемую иерархию. Избегайте использования слишком многочисленных уровней заголовков, чтобы избежать избыточной сложности в структуре текста.
