Акценты

Курсы / Markdown / Базовый синтаксис

В Markdown можно добавлять акценты (Emphasis), чтобы выделить части текста.

Давайте рассмотрим различные способы добавления акцентов.

Жирный текст

Чтобы сделать текст жирным (Bold), используйте две звездочки ** или два подчеркивания __ вокруг текста.

Пример (Markdown):

**Эл**: Я не такая, как __они__.

Результат (HTML):

<strong>Эл</strong>: Я не такая, как <strong>они</strong>.

Результат (Отображение):

Эл: Я не такая, как они.

Курсив

Чтобы сделать текст курсивным (Italic), используйте одну звездочку * или одно подчеркивание _ вокруг текста.

Пример (Markdown):

*Уилл*: Где я? Это не _Хоукинс_.

Результат (HTML): 

<em>Уилл</em>: Где я? Это не <em>Хоукинс</em>.

Результат (Отображение):

Уилл: Где я? Это не Хоукинс.

Жирный и курсив

Для комбинированного эффекта жирного и курсивного текста (Bold and Italic) используйте три звездочки *** или три подчеркивания ___ вокруг текста.

Пример (Markdown):

***Робин***: Странности снова ___начинаются___.

Результат (HTML): 

<strong><em>Робин</em></strong>: Странности снова <strong><em>начинаются</em></strong>.

Результат (Отображение):

Робин: Странности снова начинаются.

Зачеркнутый текст

Чтобы добавить зачеркивание (Strikethrough) текста, используйте две тильды ~~ вокруг текста.

Пример (Markdown):

~~Демогоргон~~: Угроза из другого мира.

Результат (HTML): 

<s>Демогоргон</s>: Угроза из другого мира.

Результат (Отображение):

Демогоргон: Угроза из другого мира.

Цветной текст

Markdown не позволяет изменять цвет текста, но если ваш процессор Markdown поддерживает HTML, вы можете использовать тег <font> HTML. Атрибут color позволяет указать цвет шрифта, используя имя цвета или шестнадцатеричный код #RRGGBB.

Пример (Markdown):

<font color="red">Угроза из другого мира.</font>

Результат (HTML): 

<font color="red">Угроза из другого мира.</font>

Результат (Отображение):

Угроза из другого мира.


Тег <font> HTML технически поддерживается, но официально устарел, что означает, что он работает в настоящее время, но вы не должны его использовать. К сожалению, нет другой чистой HTML-альтернативы. Вы можете попробовать использовать один из альтернативных вариантов с использованием встроенных стилей (Inline Styles). Не все приложения Markdown поддерживают встроенные стили (Inline Styles), но если то, которое вы используете, поддерживает, вот альтернатива тегу <font>

Пример (Markdown):

<span style="color:red">Угроза из другого мира.</span>

Результат (HTML): 

<span style="color:red">Угроза из другого мира.</span>

Результат (Отображение):

Угроза из другого мира.
