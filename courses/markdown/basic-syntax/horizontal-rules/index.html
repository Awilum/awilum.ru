Горизонтальные линии

Курсы / Markdown / Базовый синтаксис

Горизонтальные линии (Horizontal Rules) в Markdown служат для создания разделителей между различными частями текста, делая документ более читаемым и организованным.

Создание горизонтальных линий

Чтобы создать горизонтальную линию, используйте три дефиса ---, три подчеркивания ___ или три звездочки ***.

Пример (Markdown):

Глава первая: Исчезновение Уилла Байерса
		
---

Глава вторая: Чудак с Кленовой улицы

***

Глава третья: Здорово, Джолли

___

Результат (HTML):

<p>Глава первая: Исчезновение Уилла Байерса</p>
<hr>
<p>Глава вторая: Чудак с Кленовой улицы</p>
<hr>
<p>Глава третья: Здорово, Джолли</p>
<hr>

Результат (Отображение):

Глава первая: Исчезновение Уилла Байерса

Глава вторая: Чудак с Кленовой улицы

Глава третья: Здорово, Джолли
