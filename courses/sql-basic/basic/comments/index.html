Комментарии

Курсы / SQL Базовый уровень / Основы

В SQL вы можете комментировать свой код, как и в любом другом языке программирования.

Комментарии могут занимать одну или несколько строк.

Вот несколько примеров запросов с комментариями:

SELECT *
  FROM movies; -- выбрать все фильмы

/* выбрать все фильмы и вернуть только столбцы title, rating */
SELECT title,
       rating
  FROM movies;


/* 
* выбрать все фильмы
* вернуть только столбцы title, rating, release_year 
*/
SELECT title,
	   rating,
	   release_year
  FROM movies;

/* этот код закомментирован и выполняться не будет
SELECT title,
	   rating,
	   release_year
  FROM movies;
*/
